# 调用说明

可参考[API文档](https://www.volcengine.com/docs/6561/2227958?lang=zh)，大致流程如下：

1. 通过批量查询接口，查看可用音色ID（也可从控制台直接获取）；
	
2. 预付费音色：
	1. 选定音色ID后，调用Upload接口上传音频训练音色；
		
	2. 通过批量查询接口，查看音色训练状态是否已成功。**成功后还会通过version字段返回当前音色训练了几次；**
		
	3. **无需激活（启用）音色**即可以调用TTS接口，将音色ID带入voice\_type参数进行合成，进一步确认合成效果；
		
	4. 预付费音色如不满意可以再次训练，至多可训练共15次，超过时接口将返回错误；
		
	5. 满意后可以调用启用接口固定（非必须）。**注意：启用后无论之前训练过几次，都无法再次训练。** 
		
3. 后付费音色：
	1. 使用后付费音色时，不会立即收取音色槽位费用。音色槽位费用将在首次使用此复刻音色进行语音合成时收取，试听字符根据声音复刻2.0模型收费；
		
	2. 音色进行语音合成后，音色将固定，不支持再次训练复刻，所以请务必确认效果符合要求后再进行语音合成；
		
	3. 试听的音色若 7 天内未正式调用，则系统会删除该音色。
